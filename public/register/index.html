Repair Job Registration

This static page is a placeholder for the public registration flow. The API endpoint POST /api/jobs/register/:token accepts the customer details and device notes that were collected from the QR code workflow described in development_roadmap.md.

Integrate this page with your frontend framework of choice. It only needs to collect the following fields from the customer:

  • Full name
  • Phone number
  • Device type
  • Device model
  • Terms and conditions acceptance

The backend already validates the submission and links the details to the repair job that generated the QR code.
